# Connection parameters (Platform API)
This document contains all the connection parameters required for creating an [authenticated connection](/oem/oem-api/connections.md#create-connections) via Workato's platform APIs.

The privilege required to use this API endpoint is currently available to Workato OEM partners only.

## Guide

### Name

Name of the connector. Only connectors that require connections are included in this table.

### Provider

Identifier for the connector. The provider value should  be supplied while creating a connection using OEM API.

### Configuration JSON

- `oauth`: Connector requires OAuth
- `personalization`: Connection can be used as personalized connection in Workbot use cases
- `secure_tunnel`: Connection can be used to connect to on-premise systems
- `input`: Array of connection configuration fields. When empty, no configuration is required
