# Picklist parameters (Platform API)

This document contains all the connection parameters required for obtaining a list of picklist values via Workato's platform APIs.

The privilege required to use this API endpoint is currently available to Workato OEM partners only.

# Guide

# Provider

Identifier for the connector. The provider value should be supplied while creating a connection using OEM API.

# Picklist name

Internal name of the picklist.

# Parameters

If a picklist requires parameter(s), the name of the parameter is specified here. See the sample request for an example of how this is used.
